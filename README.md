

## PocketPics
Built using React, MaterialUI and FancyBox.
Reduces the number of tabs used (and thus memory too) by allowing users to paste photos into the web app.
The images are then neatly displayed using FancyBox and can be removed at any time.

Test out the app: https://pocketpics.herokuapp.com/?fbclid=IwAR2oohCwy-mMua9JJd6HqBO2aOtB0dRJI9z8vc7pdogC2RLFMANkFrJ3QI0

Planned features:
-Allow the user to submit an optional link which will be added to the caption of the images.
